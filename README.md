# 🌛 SleepWellApp
스파르타코딩클럽 팀과제

- 숙면을 도와주는 asmr영상 콘텐츠 모음 어플리케이션입니다.

# 🌛 개발 기간
- 23.09.04 ~ 23.09.09

# 🌛 필수 구현 과제 (모두 완료!)
- 로그인 화면/회원가입 화면 : UserDefaults
- 메인 페이지(영상) : API, UICollectionView, UITabBarController
- 영상 세부 페이지 : 영상 정보
- 마이 페이지 : 로그아웃, 프로필 수정

# 🌛 선택 구현 과제
- 찜하기
- 피커뷰
- placeholder

# 🌛 우리팀만의 심화 도전 과제
- 길이별로 sorting하여 영상 추천
- 자동으로 앱 종료되는 타이머
- Carousel Slide
